# Папка для авторизации

Эта папка предназначена для реализации системы авторизации.

**Задача будет выполнена другим разработчиком.**

## Структура (планируется):

```
auth/
  ├── components/        # Компоненты авторизации (LoginForm, RegisterForm)
  ├── pages/            # Страницы (Login, Register)
  ├── hooks/            # Кастомные хуки (useAuth)
  ├── services/         # Сервисы авторизации
  └── utils/            # Утилиты (token management)
```

## Требования к реализации:

- Авторизация через email/password
- Интеграция с Supabase Auth
- Защищенные роуты
- Управление состоянием авторизации через Context
- Валидация форм
- Обработка ошибок
